SOC Automation Home Lab

Description: Developed an automated Security Operations Center (SOC) in a virtual lab environment to simulate real-time incident detection and response. The setup included an Ubuntu analyst machine and a Windows 10 endpoint. Configured Wazuh (XDR/EDR) for endpoint monitoring, TheHive for incident management, and Shuffle (SOAR) for automation, utilizing Sysmon for log collection. Created rules to trigger alerts for threats like Mimikatz usage, automating email notifications to the analyst.

Key Activities:

  • Built a virtual lab with two machines to simulate a SOC environment.
  • Integrated Wazuh, TheHive, and Shuffle to streamline incident response workflows.
  • Analyzed Sysmon logs to detect and respond to simulated threats.
