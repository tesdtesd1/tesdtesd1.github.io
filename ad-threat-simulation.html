Active Directory Threat Simulation and SIEM Analysis

Description: Designed a home lab to simulate and monitor Active Directory (AD) security threats, using Windows Server (configured as an AD domain controller), Windows 10 (endpoint/victim machine), and Kali Linux (attacker machine). Deployed Splunk on an Ubuntu Server for SIEM analysis, integrating Sysmon for detailed endpoint logging. Executed a brute-force attack from Kali Linux against a domain user, analyzing telemetry in Splunk, and utilized Atomic Red Team for diverse attack simulations to evaluate SIEM effectiveness.

Key Activities:

  • Configured AD on Windows Server, promoting it to a domain controller.
  • Simulated attacks using Kali Linux and Atomic Red Team to generate telemetry.
  • Built Splunk dashboards and alerts to monitor and analyze security events.
